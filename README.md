# Auditoría de Red y Escaneo con Nmap

Este repositorio documenta el desarrollo de una auditoría perimetral sobre una serie de servidores utilizando la herramienta Nmap desde Kali Linux. Fue desarrollado para la materia de **Auditoría de Seguridad Informática** en el programa de **Técnico en Servicios de Seguridad Informática**.

## Contenido del trabajo
* **Resultados de escaneo**: Archivos HTML generados que muestran el estado de los puertos TCP/UDP de los servidores analizados.
* **Lista de objetivos**: Archivo de texto plano con los servidores que sirvieron de objetivo para el escaneo.
* **Portafolio técnico**: Informe detallado en PDF que contiene el procedimiento de análisis, explicaciones sobre protocolos de red y capturas de pantalla de la práctica.

## Archivos del repositorio
* `reports/`: Archivos HTML del reporte generado por Nmap.
* `targets/`: Lista de servidores evaluados.
* `docs/`: Documentación escolar completa de la práctica.
